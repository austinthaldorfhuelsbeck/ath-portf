Canvas Paint

Clone of MS Paint created in a 4-hour coding challenge.

Demo Site

Repo

About the project

This project was accomplished within a 4-hour timeframe. A basic set of instructions was provided, including user stories and parameters for which HTML elements should appear.

Project Execution

Breaking the problem down into smaller pieces, I decided the application would need components for a color picker, line controls, and the canvas itself. A simple json file provides data related to available colors.

The canvas element itself utilizes React Hooks including useRef, useState, and useEffect to render user input to the DOM.

Project Results

Much of the requested functionality is possible with HTML5 elements, but React brings the application to life through state and reference management. This project helped to solidify understanding of React Hooks.
