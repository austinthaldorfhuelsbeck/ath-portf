Pomodoro Timer

A React project for displaying a customizable study timer as a single page app.

Demo Site

Repo

About the project

This project appears simple, but the logic behind the timer provided an interesting challenge to display understanding of JavaScript and programming concepts.

Project Execution

This implementation handles multiple state objects and provides functions for the application to interact with the timer. The timer controls component is reused by leveraging React Hooks.

Project Results

This was my first full React app, built during Chegg Learning’s Thinkful Software Engineering frontend lessons. The approach of utilizing React Hooks to write handler functions allowed synthesis of existing JavaScript knowledge with React concepts which are important programming fundamentals.
