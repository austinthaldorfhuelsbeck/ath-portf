Study App

React Router allows users to create, edit, and delete flashcards and decks from an external API.

Repo

About the project

Utilizing a simple JSON server, this React app is designed to aid in learning through studying flashcards.

This is my submission for Chegg Learning’s Thinkful Software Engineering frontend capstone project. I was provided with a JSON database, tests, and documentation, then I was evaluated on my submission.

Project Execution

React Router provides complex functionality from even a simple API, and a challenge of this project was to handle state and API requests based on user interaction. React Router allows the user to explore and study the cards within a deck, and a breadcrumb navigation bar renders based on the user’s location.

Project Results

When first being introduced to React, it was challenging to understand how to separate an application into components and handle component props. This project put that understanding to the test, which facilitated a greater understanding of React and programming concepts in general.
